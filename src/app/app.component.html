AdHoc

¿Qué es AdHoc?

Aplicación web para facilitar la autogestión del trabajo diario de un abogado. Las funcionalidades core de esta herramienta son:

  • gestión de expedientes,
  • carga de escritos,
  • carga de clientes,
  • manejo de agenda

Visión:

La idea de AdHoc es crear una herramienta fácil de manejar y útil para gestionar los expedientes de cada cliente de un estudio de abogados o de un abogado particular. Esta herramienta facilitará no perder de vista la big picture del trabajo que se esté realizando en cada momento y organizar la información de cada cliente de manera cómoda y sencilla de navegar.
